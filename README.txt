Hello !
This is a project done as an exercise for my second year at ISART, where I had to develop a chat room with the windows Winsock API, with a server application and a client one, and a library handling the network logic.

HOW IT WORKS:
-Open the server and client application MiniCharServer.exe and MiniChatClient.exe
-Enter your username in the client app, as well as the server IP address (IPV4 or IPv6), that was given in the server app
-Upon connecting, you will receive a message with the name of all currently connected users, and a message will be sent to them, indicating them of your connection
-You can now chat with all present users
-To quit the server and close the application, type /quit then press ENTER
-To close the server and interrupt communication for all other users, type /close then press ENTER

Enjoy :D
